# Fusion 360 Design Files

This directory contains the Fusion 360 design files for the Theo Jansen Rhinoceros kinetic sculpture.

## Files

- Place your `.f3d` Fusion 360 project files here
- Include any exported design files or technical drawings

## Notes

- Fusion 360 files can be opened and edited using Autodesk Fusion 360
- Files are provided for educational and non-commercial purposes
- Design is inspired by Theo Jansen's original Strandbeest mechanisms
