## このプロジェクトについて

このリポジトリは、オランダのアーティスト **Theo Jansen（テオ・ヤンセン）** とその Strandbeest 作品に触発されたキネティック・スカルプチャーを制作する過程を記録したものです。Strandbeest（オランダ語で「浜辺の生き物」の意）は、風力で歩行する動的彫刻で、芸術、工学、自然の優雅な融合を示しています。

### 目的

- **教育的**: キネティック・スカルプチャーの設計と製作の学習プロセスを共有
- **子どもたちへのインスピレーション**: 若い学習者が工学、芸術、機械学の交差点を探求できるよう支援
- **非営利**: このプロジェクトは純粋に教育目的で作成されています
- **オープンソース**: すべての設計ファイルとドキュメントは自由に利用可能

### 帰属と敬意

このプロジェクトは、Theo Jansenのオリジナル Strandbeest メカニズムに深く触発されています。彼の革新的な仕事と芸術的ビジョンに対して多大な敬意を払っています。このプロジェクトは：

- Theo Jansenの設計の正確なレプリカ**ではありません**
- キネティック・メカニズムを理解するための学習ツールとして機能します
- **教育目的および非営利目的のみ**で共有されます
- 他の人々がTheo Jansenのオリジナル作品を研究し、鑑賞することを奨励します

**Theo Jansenの素晴らしい作品については、[公式ウェブサイト](https://www.strandbeest.com/)をご覧ください。**

## ここで見つかるもの

### 🖨️ 3Dプリント可能なパーツ
- **STLファイル**: `STL/`ディレクトリ内のプリント準備済みコンポーネント

### 📚 ドキュメント
- **組立説明書**: ステップバイステップのガイダンス（作成中）

- **バイリンガルサポート**: 英語と日本語の両方でドキュメント提供

## はじめに

### 必要なもの

- **Fusion 360**（個人/教育用として無料）
- **3Dプリンター**（または3Dプリントサービスへのアクセス）
- **基本的な工具**（組立用）

### このリポジトリの使い方

1. **設計ファイルのダウンロード**: このリポジトリをクローンするか、Fusion 360ファイルをダウンロード
2. **メカニズムの研究**: Fusion 360でファイルを開き、リンケージシステムを理解
3. **パーツの3Dプリント**: STLファイルを使用してコンポーネントをプリント
4. **組立**: ドキュメントに従って独自のキネティック・スカルプチャーを構築
5. **学習と実験**: 原理を理解するために設計を修正

## 教育目標

このプロジェクトの目的は以下を教えることです：

- **機械的リンケージ**: 接続されたパーツがどのように動きを生み出すかの理解
- **CAD設計**: 工学プロジェクトのためのFusion 360の使用方法の学習
- **3Dプリント**: 付加製造技術の実践的応用
- **問題解決**: 反復的な設計とトラブルシューティング
- **芸術+工学**: 機能的な機械芸術の美しさ
