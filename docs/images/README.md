# Images Directory

This directory contains images, photos, and diagrams for the Theo Jansen Rhinoceros project.

## Contents

Place your images here:

- Assembly photos
- Design diagrams
- Build progress pictures
- Mechanism illustrations
- Final sculpture photos

## Guidelines

- Use descriptive filenames
- PNG or JPEG format
- Reasonable file sizes (compress if needed)
- Include captions or descriptions in documentation

## Attribution

If you include photos of your build, you're welcome to add attribution here or in your pull request.
